## 1. Bối cảnh / Mục đích
Thiết kế màn hình setting user
## 2. Phạm vi
- Tham chiếu thiết kế tới các file thiết kế màn hình thiết kế tại raw/design/Setting, UI đầu ra phải giống như thư mục thiết kế này
## 3. Yêu cầu
3.1 Tận dụng lại button Settings đã tồn tại ở navigate side bar để liên kết với màn hình này
3.2 Tạn dụng Topbar, SideBar hiện tại không được phép tạo cái mới
3.3 Gia đoạn này chỉ cần dựng UI chưa cần thực hiện bất cứ thao tác nào khác
3.4 Khi nhấn vào biểu tượng avartar có thể cho thay đổi hình ảnh đại diện của user, hình ảnh này sẽ được upload từ local và có cơ chể lưu lại ở server
3.5 Có thể thay đổi password hiện tại
3.6 Có thể chỉnh giao diện sáng tối
3.7 Có thể đăng xuất khỏi tài khoản
3.8 Nhấn nút Luu thay doi thì sẽ lưu lại tất cả các cài đặt này
